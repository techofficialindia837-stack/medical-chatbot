# 🩺 Medical Q&A Chatbot

## 📌 Overview
This project is a Medical Question Answering Chatbot built using the MedQuAD dataset. It retrieves the most relevant answer based on user queries.

## 🚀 Features
- Medical question answering system
- TF-IDF based retrieval
- Cosine similarity matching
- Basic medical entity recognition
- Streamlit-based UI

## 🧠 Tech Stack
- Python
- Scikit-learn
- Streamlit

## 📂 Dataset
MedQuAD Dataset (Medical Question Answer Dataset)

## ⚙️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
python -m streamlit run app.py

## 🎯 Output
The chatbot takes a medical question and returns the most relevant answer along with detected entities.
